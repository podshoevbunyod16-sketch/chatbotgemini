Деплой на Render за 3 шага

  1. Загрузите этот проект в репозиторий GitHub (включите app.py, requirements.txt, папку templates)
  2. На Render.com выберите "New Web Service" → подключите репозиторий → указав: Build Command: pip install -r requirements.txt и Start Command: gunicorn app:app
  3. Нажмите "Create Web Service" — готово! Сайт будет доступен 24/7
📌 Проверка health: /health → {"status": "healthy"}
